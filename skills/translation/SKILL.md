# Translation Guidelines

## Target Language

Translate to **Traditional Chinese** (繁體中文).

## Technical Terms

When translating technical terms, keep the English original in parentheses after the translated term.

### Format

```
[中文翻譯] ([English original])
```

### Examples

- 向量暫存器檔案 (vector register file)
- 暫存器壓力 (register pressure)
- 編譯器 (compiler)
- 連結器 (linker)
- 堆疊 (stack)
- 堆積 (heap)
- 指標 (pointer)
- 參考 (reference)
- 執行緒 (thread)
- 行程 (process)

## Consistency

- Use consistent translations for the same term throughout a document
- Prefer commonly accepted Traditional Chinese technical terminology
- When uncertain, prioritize clarity over literal translation
